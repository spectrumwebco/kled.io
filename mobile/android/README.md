# Kled.io Android App

This directory contains the Android application for Kled.io, built using React Native with the Lynx framework.

## Getting Started

### Prerequisites

- Android Studio
- Node.js 16 or later
- JDK 11

### Installation

1. Install dependencies:

```bash
yarn install
```

2. Run the app:

```bash
yarn android
```

## Features

- Workspace management
- Cluster management
- Policy management
- Real-time monitoring
- Push notifications

## Building for Production

```bash
yarn build:android
```
