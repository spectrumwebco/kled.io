# Kled.io iOS App

This directory contains the iOS application for Kled.io, built using React Native with the Lynx framework.

## Getting Started

### Prerequisites

- Xcode 14 or later
- Node.js 16 or later
- CocoaPods

### Installation

1. Install dependencies:

```bash
yarn install
```

2. Install CocoaPods dependencies:

```bash
cd ios
pod install
cd ..
```

3. Run the app:

```bash
yarn ios
```

## Features

- Workspace management
- Cluster management
- Policy management
- Real-time monitoring
- Push notifications

## Building for Production

```bash
yarn build:ios
```
